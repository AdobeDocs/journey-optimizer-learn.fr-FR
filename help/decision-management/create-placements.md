# Création d’emplacements

Découvrez comment créer des [emplacements](https://experienceleague.adobe.com/docs/journey-optimizer/using/offer-decisioniong/create-components/creating-placements.html?lang=fr) dans [!DNL Offer Decisioning]. Les emplacements sont l’un des composants de base des offres.
Un emplacement représente la combinaison du type de contenu et du canal, par exemple une image dans un email ou du code HTML sur un site Web.

>[!VIDEO](https://video.tv.adobe.com/v/329372?quality=12&learn=on)

Pour en savoir plus sur la création d’emplacements à lʼaide dʼune API, consultez le [guide du produit](https://experienceleague.adobe.com/docs/journey-optimizer/using/offer-decisioniong/api-reference/offers-api/placements/create.html?lang=fr).
