# Créer un jeu de données de recherche

Une recherche de jeu de données permet à Adobe Journey Optimizer de récupérer des données de référence ou transactionnelles d’un jeu de données Adobe Experience Platform au moment de l’exécution du parcours, sans avoir à stocker ces données sur le profil client en temps réel.

Dans ce cas d’utilisation, lorsqu’un utilisateur ou une utilisatrice envoie un formulaire de préférences d’investissement (actions, obligations ou CD), le parcours recherche dynamiquement les détails d’investissement correspondants à partir d’un jeu de données de référence et utilise ces informations pour personnaliser un e-mail.
