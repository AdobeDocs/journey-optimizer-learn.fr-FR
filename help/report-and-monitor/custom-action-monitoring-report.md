# Surveiller vos actions personnalisées

Découvrez comment surveiller efficacement l’intégrité opérationnelle et les performances des appels HTTP effectués à partir de vos parcours vers des points d’entrée tiers au moyen d’actions personnalisées. Gagnez en visibilité sur les erreurs HTTP, les goulets d’étranglement liés à la latence et les problèmes de limitation qui peuvent affecter la diffusion.

Ce rapport permet d’identifier les problèmes, d’améliorer les performances de parcours et d’assurer un fonctionnement fluide.

>[!VIDEO](https://video.tv.adobe.com/v/3479541/?learn=on&enablevpops)

Pour plus d’informations sur cette fonctionnalité, voir la [documentation du produit](https://experienceleague.adobe.com/fr/docs/journey-optimizer/using/configure-journeys/action-journeys/reporting).
