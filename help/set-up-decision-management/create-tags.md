# Création de balises

Découvrez comment [créer des balises](https://experienceleague.adobe.com/docs/journey-optimizer/using/offer-decisioniong/create-components/creating-tags.html?lang=fr) in [!DNL Offer Decisioning]. Les balises sont des composants de blocs de création d’offres facultatifs. Elles peuvent être utilisées pour organiser les offres et les regrouper dans des collections dynamiques.
